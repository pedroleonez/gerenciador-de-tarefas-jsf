Sobre - Gerenciador de Tarefas

Sobre a Aplicação

Este Gerenciador de Tarefas foi desenvolvido como parte de um processo seletivo para demonstrar conhecimentos em Java, JSF, PrimeFaces e PostgreSQL.

Arquitetura

A aplicação utiliza o padrão de camadas (Entity, Repository, Service, Controller) para garantir uma separação clara de responsabilidades.
